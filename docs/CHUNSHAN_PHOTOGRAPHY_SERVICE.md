## 5. UI/UX 要點（可套用至本專案）

1. **分類導覽**：以 Tab 或篩選列切換「優惠活動、組合套餐、職業形象照…」等，同一頁可顯示多分類，或依分類篩選產品卡。
2. **產品卡**：標題 + 原價（可選）+ 售價 + 一句描述 + 主按鈕（SEE MORE／立即使用）。
3. **限定標籤**：部分方案顯示「台北、勤美店限定」「每日限 2 組」等，可用 badge 呈現。
4. **價格區間**：支援單一價格或區間（如 NT $3800~7900）。
5. **作品集區**：可另設作品相冊（領袖之境、爾雅界、主題寫真系列等）作為視覺參考。

## 6. 與 enhance-pixshop 的對應構想

| 春山服務類型 | 本專案現有／可對應功能 |
|--------------|------------------------|
| 韓式證件照、證件照＋形象照 | 現有 **IdPhoto** 流程（證件照規格、完顏等級等） |
| 主題寫真、場景風格 | 現有 **Travel** 場景寫真；可擴充「主題風格」選項 |
| 職業形象照、肖像照 | 可為「形象照」新流程或沿用編輯器（修圖/濾鏡/裁切） |
| 雙人／多人寫真 | 目前多為單人；可規劃為「多人／團體」擴充或說明文案 |
| 優惠活動、組合套餐 | 可做成「方案列表」或「推薦組合」區塊，連結至對應流程 |

以上結構與清單可供本專案設計「攝影服務／服務項目」頁面與資料結構時直接參考或簡化使用。
